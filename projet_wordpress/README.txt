Ces sources sont configurées pour une base de données nommée wordpress, se trouvant sur le serveur localhost, à laquelle se connecte l'utilisateur root avec le mot de passe troiswa.
Si ce n'est pas le cas chez vous, modifiez le fichier wp-config.php

L'URL du site est http://localhost/wordpress
Si ce n'est pas le cas chez vous, appliquez un script de migration sur la BDD
